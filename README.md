# 📊 Store Data Analysis Dashboard (Excel)

## 📌 Project Overview

This project focuses on analyzing store performance using an interactive Excel dashboard. It provides insights into sales trends, customer behavior, and channel performance to support data-driven decision-making.

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Visualizations
* Slicers & Filters

## 🎯 Objectives

* Analyze overall sales and order performance
* Understand customer distribution across demographics
* Identify top-performing sales channels
* Track monthly sales trends and order status

## 📊 Dashboard Features

* Interactive filters using slicers (Gender, Channel, etc.)
* Dynamic charts for real-time data exploration
* Clean and user-friendly layout for business insights

## 🔍 Key Analysis Performed

* **Sales Analysis:** Compared total sales and order volume across different periods
* **Customer Insights:** Analyzed customer distribution based on gender and age group
* **Channel Performance:** Evaluated sales contribution from platforms like Amazon, Flipkart, Myntra, etc.
* **Geographical Analysis:** Studied sales distribution across different states
* **Order Status Analysis:** Monitored order statuses such as delivered, cancelled, and returned
* **Trend Analysis:** Identified monthly sales patterns and seasonal trends

## 🚀 Key Insights

* Identified top-performing sales channels contributing maximum revenue
* Observed customer buying patterns across different demographics
* Detected seasonal trends in monthly sales data
* Highlighted regions with highest and lowest sales performance
